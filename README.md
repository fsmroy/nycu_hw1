# nycu_hw1

統計學小組作業。

## 專案說明

本專案整理兩個台灣市場標的的歷史資料，並以月報酬率作為後續統計分析的主要基礎：

- 元大台灣 50 ETF（0050）
- 臺灣加權股價指數

分析期間為 2018/01/01 至 2026/06/30。

## 資料來源

資料是使用 Python 套件 yfinance 抓取 Yahoo Finance 的歷史資料。

- 0050 代號：0050.TW
- 臺灣加權指數代號：^TWII

對應的 Yahoo Finance 頁面可參考：

- 0050 歷史資料：https://hk.finance.yahoo.com/quote/0050.TW/history/
- 臺灣加權指數歷史資料：https://finance.yahoo.com/quote/%5ETWII/history/

實作上是透過 yfinance 下載日資料，再整理成專案內需要的 CSV 格式。

## 原始日資料如何整理

最初下載的日資料包含日期與收盤價等欄位，後續為了讓分析重點更明確，整理成只保留：

- 日期
- 收盤價

接著再把 0050 與臺灣加權指數兩份資料依日期對齊，產出比較檔。

## 為什麼只保留每月最後一天

本專案後續分析的核心是月報酬率，因此不直接使用每日收盤價，而是先從日資料中取出每個月份最後一個交易日的收盤價。

注意這裡的「每月最後一天」指的是：

- 該月最後一個有交易的日期
- 不一定是曆月的最後一天

例如如果月底遇到假日或休市，就會使用該月最後一個交易日。

## 月報酬率如何計算

後續所有統計分析以月報酬率為主，而不是直接分析收盤價。

月報酬率公式如下：

月報酬率 = （本月月底價格 − 上月月底價格）／上月月底價格

例如：

- 2024/01 月底價格 = 180
- 2024/02 月底價格 = 186

則 2024/02 的月報酬率為：

（186 − 180）／180 = 0.0333 = 3.33%

在程式計算上，做法等同於：

- 先找出每個月最後一個交易日的收盤價
- 將本月收盤價與上月收盤價相比
- 以前月價格為分母計算變動比例

因此月報酬率檔中的第一筆資料會是空值，因為第一個月份沒有前一個月可以比較。

## 月報酬率檔案欄位說明

[data/0050_台灣加權指數_月報酬率比較.csv](data/0050_台灣加權指數_月報酬率比較.csv) 的欄位如下：

- 日期：每月最後一個交易日
- 0050月報酬率：0050 相對於前一個月月底收盤價的報酬率
- 臺灣加權指數月報酬率：臺灣加權指數相對於前一個月月底收盤價的報酬率

檔案中的月報酬率目前以小數表示：

- 0.05 代表 5%
- -0.02 代表 -2%
